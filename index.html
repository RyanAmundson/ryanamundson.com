Whats that Word?

"Whats that Word?" is an Amazon Alexa Skill which provides the user the capability to describe a word or give a synonym of a word they may have forgotten. Using the OneSearch and Datamuse API Alexa will attempt to guess the forgotten word using a reverse dictionary search and natural language processing algorithms. Written in NodeJS and uses AWS event driven lambda function to interface with Alexa.

Command Aviation Database

This project is a desktop application built for Command Aviation which is a small plane refueling company located at the Bellingham Airport. The application is written in Java and utilizes javaFX 8 for the user interface. The main use is creating reservations for incoming aircraft and easily moving clients through the servicing process all while visually displaying the current state of each client. Built on a MySQL back-end.

Deadwood Studios

Deadwood studios is a board game developed by KickAss Games. It is a fun turn based monopoly style game in which you are a struggling actor trying to make it big. The game puts you in a Hollywood backlot where you must act and compete to make cash and credit.
